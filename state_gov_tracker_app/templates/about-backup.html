What is StateRep.Me?

StateRep.Me was conceived and developed during Hacks4Democracy, a Fall 2012 Azavea-sponsored civic hackathon in Philadelphia, PA. A collaborative project from Jason Blanchard, Josh Borkowski, Christopher Brown, Joshua Darr, Lauren Gilchrist, Christopher Nies, Andrew Thompson, and Nickolas Weingartner, StateRep.me compiles the votes, press releases, tweets, and Facebook posts of all Pennsylvania state legislators in one simple web portal to enable easy tracking of legislative actions by individual representatives. Searchable by address, the goal of the application is to increase legislator accountability and constituent engagement by reducing the time and effort it takes to monitor the Pennsylvania state legislature.

About the StateRep.Me Data

StateRep.Me is constructed using a combination of publicly available and unique web-scraped data:

Cicero
Azavea’s Cicero API, a custom geocoder that matches addresses to legislative districts and elected official information, powers StateRep.Me’s address search functionality. The StateRep.Me team has also used it to develop lists of Twitter and Facebook handles for Pennsylvania representatives.
Open States
A project of the Sunlight Foundation, the Open States API powers StateRep.Me's list of Pennsylvania state legislators, including contact information and voting history.
Press Releases
The StateRep.Me Team gathers press releases from each legislator's website to generate the list of press releases. In the spirit of open data, StateRep.Me recently released these data as an API.
Ideology Scoring
Calculated using a Bayesian item response model developed by political scientists Simon Jackman, Joshua Clinton, and Douglas Rivers using the voting history of each legislator. To estimate preferences/ideology, this model uses differences in roll call voting behavior in the Pennsylvania General Assembly to scale legislators on a single dimension (the left-to-right political axis). This method is preferable over other measures that only use a subjective number or list of roll call votes to create an ideology score for each member based on a subset of votes and is the same method used to measure ideology for members of Congress and Supreme Court Justices in the political science literature.
Facebook Posts
Using Facebook's API, StateRep.Me collects each legislator's Facebook posts in near-real time.
Tweets
The Twitter API powers StateRep.Me's PA Tweets page as well as individual representatives' Twitter updates.
